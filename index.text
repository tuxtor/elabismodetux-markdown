Bash cheats
===========
Search recursively over files into a directory 

	find . -type f -exec grep '<String or regepx>' {} \; -print 
***** 
Add an alias for git add&commit&push (one-shot git!)
	git config --global alias.ons '!git add -A && git commit && git push'
